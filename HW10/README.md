#########################################################
# LCS
#########################################################

Longest common subsequence

Даны две строки.\
Найти самую длинную последовательность символов, которая встречается в заданном порядке в обеих строках. Символы не обязательно должны идти подряд\

Пример:
string_1 = "AGGTAB"\
string_2 = "GXTXAYB"\

LCS = "GTAB"\

**Важно!**
Тесты, в рамках которых необходимо рассмотреть как можно больше краевых кейсов.
