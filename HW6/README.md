# K-th
#########################################################

Найти k-ый по величине элемент массива.\

**Важно!**
Нельзя просто отсортировать массив. Также не используем кучи, так как про них еще пока не знаем.\
**Подсказка:** quickselect.\
Тесты, в рамках которых необходимо рассмотреть как можно больше краевых кейсов. 

Пример 1:\

Ввод: nums = [3,2,1,5,6,4], k = 2\
Вывод: 5

Пример 2:\

Ввод: nums = [3,2,3,1,2,4,5,5,6], k = 4\
Вывод: 4
